# RepRepo Website
This is the web profile for Daniel Griffiths

### Application Layer Stack
- Typescript
- Vue 3

### Operations Stack
- AWS
- CI/CD with GitHub Actions

### Initialization
- `pnpm i`
- `pnpm serve`

### Deploy
- `docker build -t reprepo .`
- `docker run --name reprepo -dp 127.0.0.1:8080:8080 reprepo`
